# 🌌 SYSTEM_CORE: OBSIDIAN_EDITION

Welcome to the **Obsidian Edition** of my professional portfolio. This is a high-performance, kinetic, and immersive digital experience built for **Soyam Kapasiya**, a High-Performance Backend Architect.

> "Engineering the backplane of tomorrow's intelligent systems."

## 🚀 SYSTEM_FEATURES

### 1. 🌑 Kinetic Minimalism
A premium, monochromatic theme using **Obsidian Core** colors with **Hyper-Cyan** accents. Designed for high visual impact and a "Mission Critical" aesthetic.

### 2. 🕹️ Interactive Defense Protocol (Mini-Game)
The landing page features a real-time **Rogue AI Defense** system. Integrated directly into the DOM, it allows users to intercept rogue data packets using a specialized fluid cursor.

### 3. 📄 Multi-Page Architecture
A robust, modular structure designed for scale:
- `/index.html`: The **Biometrics Hub** (Landing + Mini-Game).
- `/expertise.html`: The **Kernel** (Deep-dive into Logic, Intelligence, and Persistence).
- `/archives.html`: The **Deplayment Archives** (Project showcase).
- `/uplink.html`: The **Secure Communications** (Contact terminal).

### 4. ⚡ Intelligent Interactions
- **Fluid Custom Cursor**: Scales and reacts to environmental interactables.
- **Intersection Observers**: Smooth kinetic reveals for content components.
- **Bento Grid 2.0**: Asymmetrical layout for technical mastery.
- **Hyper Marquee**: Infinite-scroll technology reel.

## 🛠️ THE_KERNEL: TECH_STACK

- **Logic**: Java 8/11/17, Spring Boot, FastAPI, Python
- **Intelligence**: RAG Architecture, LLM Orchestration, pgvector, LangChain
- **Persistence**: PostgreSQL, Snowflake, Oracle, MongoDB
- **Automation**: Temporal.io, Jenkins, Docker, CI/CD
- **Frontend**: Vanilla HTML5, CSS3 (Modern Utilities), JavaScript (ES6+)
